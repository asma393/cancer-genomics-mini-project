# Cancer Genomics Mini Project

This project demonstrates a simple bioinformatics pipeline for analyzing tumor mutation datasets using Linux command-line tools and Bash scripting.

## Project structure

Cancer_Genomics_Mini_Project
│
├── data
│   └── tumor_mutations.txt
│
├── scripts
│   └── cancer_pipeline.sh
│
├── results
│   ├── gene_counts.txt
│   ├── mutation_types.txt
│   ├── mutation_spectrum.txt
│   └── chromosome_distribution.txt

## Analysis steps

1. Count mutations by gene
2. Analyze mutation types
3. Identify mutation spectrum
4. Analyze chromosome distribution

## Tools used

- Linux command line
- Bash scripting
- grep
- cut
- sort
- uniq
